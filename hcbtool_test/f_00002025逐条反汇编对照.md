## 4. 关键观察

- `0x01 00 00` 对应 `init_stack args=0 locals=0`。
- `0x0C` / `0x0B` 分别用于压入 i8 / i16 整数；例如 `0C 01` 是 `S0 = 1`，`0B FF 00` 是 `255`。
- `0x1A` 是 `add`，`0x1B` 是 `sub`；Lua 中的 `S1 = (S1 - S2)`、`S2 = (S2 + S3)` 均来自这类栈顶二元运算。
- `0x03 07 00` 是 syscall 7，即 `ColorSet`，固定消耗 5 个参数。
- 末尾 `0x04 0x04` 是两个连续 `ret`，对应 Lua 中 block 0 的 `return` 与 block 1 的 `return`。
